# Sweet Shop Management System (Minimal - Assessment)

This is a minimal, ready-to-run submission for the Sweet Shop Management System kata.
Stack:
- Backend: Node.js + Express + SQLite
- Tests: Jest + Supertest (TDD)
- Frontend: React + Vite (SPA)

## Quick start

### Backend
cd backend
npm install
mkdir -p src/data
npm test       # run tests (creates DB file)
npm start      # runs server on http://localhost:4000

### Frontend
cd frontend
npm install
npm run dev    # open the Vite dev server (http://localhost:5173)
